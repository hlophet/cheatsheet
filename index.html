Cheat Sheet

Learning HTML and/or CSS can be such a daunting task but hopefully this cheat sheet can serve as a guide for supporting the development of future projects. And as such, this site can provide much needed relieve whenever one is stuck on a specific problem.

HMTL Semantic Elements

HTML (HyperText Markup Language) Semantic elements are tags that clearly describe their meaning and the type of content they contain, both to the browser and to the developer. Unlike non-semantic elements like <div> and <span>, which are generic containers without inherent meaning, semantic elements add context and structure to a webpage.

Tag Name Description
<article> Atricle Defines a self-contained block of content that can exist in any context.
<aside> Aside Defines a block of content that is related to the main content.
<figcaption> Figcaption Defines the caption of a <figure>.
<figure> Figure Defines a single self-contained element, usually an image.
<footer> Footer Defines the footer of a web page or section.
<header> Header Defines the header of a web page or section.
<main> Main Defines the main content of a web page. Can be displayed with a sidebar.
<mark> Mark Defines highlighted text.
<nav> Nav Defines a section with navigation links.
<section> Section Defines a section within a web page.
<time> Time Defines a time on a 24h clock.

CSS Properties

CSS (Cascading Style Sheets) properties are used to control the styling and layout of HTML elements on a web page. Without CSS, a webpage will appear bland with only text and images on a white background. Thanks to CSS, we can build websites that are aesthetically pleasing. Each property defines a specific visual characteristic or behavior of an element.

Property Description
color: Sets the text color.
opacity: Controls the transparency of an element.
background-color: Sets the background color of an element.
background-image: Specifies a background image.
float: Positions an element to the left or right, allowing text to wrap around it.
position: Specifies the positioning method (e.g., static, relative, absolute, fixed).
disaplay: Determines how an element is rendered (e.g., block, inline, flex, grid).
border: Defines the border around an element.
padding: Creates space between an element's content and its border.
margin: Creates space outside an element's border.
height: / width: Set the dimensions of an element.
text-decoration: Adds or removes lines (e.g., underline, none).
text-align: Aligns text horizontally (e.g., left, center, right).
font-size: Adjusts the size of the text.
font-family: Specifies the font typeface.
